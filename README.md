## Requirements
- hugo **with extras**

## Installation

- Build and install hugo extended
```
go build -o /tmp/hugo -tags extended
sudo mv /tmp/hugo /usr/local/bin/hugo
```

- Install `npm`
- Install dependencies
    ```
    npm install
    ```

- Build favicon bundle
    ```
    make favicon
    ```
